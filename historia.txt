Experimiento 2

Vamos a practicar usando git stash y git cherry-pick.
git stash guarda los avances hasta el momento, sin necesidad de hacer un commit, lo cual los deja disponibles para ser agregados a otro commit de otra rama.
git cherry-pick permite traer un commit expecifico de cualquier rama y agregarlo al commit actual.
